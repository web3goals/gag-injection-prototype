# Agent

### Commands

- Install dependencies - `npm install`
- Start the agent - `npm run start`
- Start a screen session - `screen -S agent`
- Detach from a session - `Press Ctrl + A, then D`
- List sessions - `screen -ls`
- Reattach session - `screen -r agent`

## Example of `.env`

```
OPENAI_API_KEY=""
NEYNAR_API_KEY=""
AGENT_SIGNER_UUID=""
PINATA_JWT=""
AGENT_PRIVATE_KEY=""
MONGODB_URI=""
```
